# Create WWTP Influence Map

Create map showing WWTP contribution as proportion of total nutrient
load

## Usage

``` r
map_wwtp_influence(data, nutrient, title)
```

## Arguments

- data:

  sf object. Integrated data with WWTP proportions

- nutrient:

  Character. "nitrogen" or "phosphorus"

- title:

  Character. Map title

## Value

ggplot object
