# Create Agricultural Classification Map

Create map showing agricultural nutrient classifications

## Usage

``` r
map_agricultural_classification(data, nutrient, classification_col, title)
```

## Arguments

- data:

  sf object. Spatial data with classifications

- nutrient:

  Character. "nitrogen" or "phosphorus"

- classification_col:

  Character. Name of classification column

- title:

  Character. Map title

## Value

ggplot object
