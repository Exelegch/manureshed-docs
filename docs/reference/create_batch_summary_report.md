# Create Batch Summary Report

Generate comprehensive summary report for batch analysis

## Usage

``` r
create_batch_summary_report(
  batch_results,
  years,
  nutrients,
  output_dir,
  verbose
)
```

## Arguments

- batch_results:

  List of results

- years:

  Vector of years

- nutrients:

  Vector of nutrients

- output_dir:

  Output directory

- verbose:

  Show progress

## Value

Summary data frame
